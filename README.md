## Deployment

To deploy this project run

```bash
  netlify deploy --prod
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/CodeinIP/U3_sportsman_clone.git
```

Go to the project directory

```bash
  cd U3_sportsman_clone
```

Start the server

```bash
  use live server
```
